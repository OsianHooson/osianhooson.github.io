Lifeform Sequencer

A physics-based generative music sequencer iOS app.

About

This is an ongoing solo project I'm working on to teach myself iOS development and generative music making. It's loosely inspired by the mutation and crossover functions from genetic algorithms. When lifeforms collide they exchange part of their musical "DNA".

The interface and underlying systems are in place. I've done some testing with the AudioKit framework. The next step is to start work on the audio engine.

Tools

This project is built in Xcode with SpriteKit, UIKit, AudioKit and CoreMotion.
